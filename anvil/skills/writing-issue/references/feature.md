# Feature issue — body guidance

A feature issue proposes a new capability. Nothing exists to reproduce yet — do **not** author a `reproduction_anchor` (forcing one is a category error).

## Body shape

- **## Problem** — the user value and the gap: who is blocked, what they cannot do today, why it matters. State the success criteria — the observable outcome that means the feature delivered.
- **## Non-goals** — the scope boundary from the smallest-viable frame: what this feature deliberately does not cover.
- **## Verification** — Direct exercises the new capability via tests; Indirect invokes it on the built artifact and asserts the user-visible outcome (see `docs/issue-spec.md`).
